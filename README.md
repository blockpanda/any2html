# any2html - 甲子渡口高级数据图文卡片排版技能

`any2html` 是一个专为 AI Agent（如 Gemini CLI、Claude 等）设计的高级前端排版技能（Skill）。它能将用户提供的长文本、生硬的数据表格、或是复杂的文章链接，经过深度信息提炼后，转化为**极具复古美感、视觉舒适、纯静态**的高质量 HTML 信息卡片。

无论是用来做小红书轮播图、推特长图、还是公众号文章配图，`any2html` 都能帮你自动化完成从“干货提炼”到“设计排版”的完整闭环。

---

## 🌟 核心特性与工作流

本技能内置了严格的**“两步走（强制前置审阅）”**工作流，彻底解决 AI 盲目排版导致的数据丢失和幻觉问题：

1. **第一步：深度提炼与排版预告（大纲输出）**
   - AI 会剔除废话，直击核心判断，强制提取所有具体数字、倍率和对比关系。
   - 遇到硬核干货时，**绝不压缩指标**，而是通过预估容量自动规划多页卡片。
   - 若存在 3 个以上的对比数据，自动触发 ECharts 静态图表绘制计划。
   - **输出标准化大纲等待你的审核。**
2. **第二步：纯静态海报生成（审核通过后直出）**
   - 你确认大纲无误后，AI 会严格按照大纲生成带有 Tailwind CSS 和 ECharts 静态配置的单文件 HTML。
   - 严格执行断层拦截、防文字溢出和复古配色，确保出图即成品。

## 📏 支持的三大排版尺寸与适用场景

在启动任务时，AI 会先询问或你可以直接指定使用哪种尺寸。不同的尺寸对应不同的信息分发场景：

### 1. 竖版（小红书多图版）
- **尺寸**：固定宽度 `1080px`，高度锁定 `1440px` (比例 3:4)。
- **特点**：单页容量有限，遇到长文会自动生成**多个卡片模块**（多页排版），并带有精美的页脚编号（如 01 / 03）。
- **适用场景**：极度适合作为**小红书图文笔记**的滑动卡片，手机端连续阅读体验最佳。

### 2. 横版（一图流全览版）
- **尺寸**：固定宽度 `1280px`，高度自适应伸缩（基础 16:9，最低 `720px`）。
- **特点**：没有高度上限，所有的内容、图表、数据都会在一张图里流畅铺展到底。
- **适用场景**：适合将长文转为一张超长的逻辑信息图。非常适合发在 **X (Twitter)**，或者作为**小红书的“小绿书（单张长图）”**形式发布。

### 3. 插图版（文章极简配图版）
- **尺寸**：绝对固定在 `1280px * 720px`。
- **特点**：极致克制。去除了所有冗余的关键词标签、TLDR 阅读时长和页码。主内容区仅保留提炼出的标题与核心金句。
- **适用场景**：专门用于公众号、博客、Notion 文章的**首图 / 封面图 / 题图**，提升文章整体的高级感。

## 🛠 如何定制适配为你自己的品牌风格？

如果你想将“甲子渡口”的视觉风格替换为你自己的品牌，只需打开本技能的 `SKILL.md` 文件，修改其中的 **HTML 骨架模板** 和 **调色盘规则**：

### 1. 替换品牌 Logo 和名称
在 `SKILL.md` 的 `<!-- 主内容区 -->` 上方，找到以下代码并替换为你自己的名称与首字母：
```html
<div class="flex items-center">
    <!-- 修改此处的首字母和背景色 -->
    <div class="bg-[#C88851] text-[#F4EFE6] font-serif font-black text-2xl w-10 h-10 flex items-center justify-center mr-3">甲</div>
    <!-- 修改此处的品牌名称 -->
    <div class="text-2xl font-serif font-black tracking-widest text-[#1A1814]">子渡口</div>
</div>
```
同时，记得修改右下角的落款署名（搜索 `@半大熊猫` 并替换）。

### 2. 修改全局字体
默认使用了 Google Fonts 提供的 `Noto Sans SC`（黑体）和 `Noto Serif SC`（宋体/复古标题）。
如果你有本地字体，可以修改 `SKILL.md` 中的 `<style>` 标签，或者保留目前的优雅搭配。

### 3. 调整品牌专属调色盘
在 `SKILL.md` 中搜索 `## 扩展重点调色盘`，你可以直接把颜色的 HEX 值替换为你的品牌色。目前系统使用的复古色系如下（可替换）：
- **背景主色**：`#E6E2D8`（复古灰白外层）, `#F4EFE6`（羊皮纸色卡面）
- **点睛主色（复古红）**：`#B7332C`（用于重要数据高亮和 TLDR）
- **沉稳辅助色（暗夜绿）**：`#1E3027`（用于模块序号和标签背景）

### 4. 调整图表颜色预设
在指令的“数据可视化（ECharts 静态海报模式）”段落中，明确要求了 AI 必须使用【扩展重点调色盘】中的颜色来绘制图表。当你修改了上述的调色盘后，AI 自动生成的 ECharts 柱状图/饼图也会自然适配你的新品牌色。

## 🚀 安装与使用

1. 将本项目整个目录克隆/下载到你的 AI Agent 技能文件夹（如 `~/.gemini/skills/any2html/`）。
2. 在对话中输入触发词：`帮我把这个链接排版成卡片`、`生成复古海报`、`用 any2html 处理这段数据`。
3. AI 会自动进入第一阶段，输出大纲给你审核。
4. 审核确认后，AI 会在本地 `/tmp` 或指定目录生成 `.html` 文件并自动在浏览器中打开，你只需右键截图或使用相关插件保存为图片即可发布。
